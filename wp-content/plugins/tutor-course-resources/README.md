# Tutor Course Resources

Plugin de WordPress para gestionar recursos de curso con Google Drive y archivos físicos para Tutor LMS.

## Características

- ✅ Gestión de recursos de curso (Google Drive y archivos físicos)
- ✅ Control de acceso por roles (Estudiantes, Docentes, Administradores)
- ✅ Interfaz de administración completa
- ✅ Integración con Tutor LMS
- ✅ Soporte para todos los tipos de archivo
- ✅ Notificaciones por email a estudiantes
- ✅ Meta box en edición de curso

## Instalación

1. Sube la carpeta `tutor-course-resources` a `/wp-content/plugins/`
2. Activa el plugin desde el panel de administración de WordPress
3. Asegúrate de tener Tutor LMS instalado y activo

## Uso

### Para Docentes

1. Ve a **Recursos de Cursos** en el menú de administración
2. Haz clic en **Añadir Recurso**
3. Selecciona el curso y el tipo de recurso (Google Drive o Archivo)
4. Configura los permisos de acceso
5. Guarda el recurso

También puedes agregar recursos directamente desde la edición del curso usando el meta box "Recursos del Curso".

### Para Estudiantes

Los recursos aparecerán automáticamente en las páginas de curso y lecciones según los permisos configurados.

## Requisitos

- WordPress 5.0+
- PHP 7.2+
- Tutor LMS (instalado y activo)
